## React Redux messages test

Simple message manager with react ecosystem

### Project CLI

```sh
# install packages
yarn

# run project
yarn start

# run storybook (Storybook server start on http://localhost:9009/)
yarn storybook

# run test
yarn test

# run flow type
yarn flow
```

### Tech Stack

* React
* Redux
* Flow
* Storybook
* Jest
* Fela
* Ramda
* Moment
